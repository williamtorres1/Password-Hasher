### Installing

```shell
# Clone
$ git clone git@github.com:williamtorres1/password-hasher.git

# Access the folder
$ cd password-hasher

# Install the dependencies
$ yarn

# Run
$ yarn dev

# The server will run on port 3333 - <http://localhost:3333

```
